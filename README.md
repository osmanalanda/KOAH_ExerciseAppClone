KOAH Exercise App Clone
A pixel-perfect, fully functional mobile application clone designed for COPD (KOAH) patient exercises and health tracking. This project focuses on providing a user-friendly interface for exercise demonstrations, blog content, and administrative management.

🚀 Features
User Authentication: Secure and simple email/password-based login and registration system.

Exercise Library: Detailed list of exercises specifically curated for COPD patients, featuring functional video playback for proper technique demonstration.

Blog Section: Educational content and articles to help users manage their health effectively.

Admin Dashboard: A dedicated management interface accessible only to users with "admin" privileges, allowing for content updates and user management.

Responsive UI: Developed with a focus on high-fidelity design, matching the original application's look and feel precisely.

Turkish Language Support: The entire user interface and content are localized in Turkish for target audience accessibility.

🛠️ Tech Stack
Framework: React Native with Expo

Styling: Tailwind CSS via NativeWind

State & Data Management: Optimized with a modular structure, currently utilizing local mock data (e.g., mockData.js) with a schema designed for seamless MongoDB API integration.

Media: Integrated video player for exercise walkthroughs.
