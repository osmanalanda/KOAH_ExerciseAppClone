# KOAH Egzersiz - COPD Exercise App Clone


This is a Turkish COPD (KOAH) exercise app for patients, featuring exercise programs, an information/consent form, and an admin dashboard. The design uses a bold green header and clean white content areas.

**Features**
- Browse exercise categories: Warm-up (Isınma), Breathing Exercises (Nefes Egzersizi), and Strengthening Exercises
- View exercise details with images, descriptions, and instructions
- "Are you ready?" confirmation dialog before starting exercises
- Read the informed consent / information form about the study
- Admin control panel showing stats (total videos, blog posts, users, support requests)
- User profile page with account info
- "Son Egzersiz" (End Exercise) button to finish a session
- Navigation between exercises (morning/evening sessions)

**Design**
- Bold green (#0D9B6A) header bar with "KOAH EGZERSİZ" title in white
- User profile and lock icons in the top-right corner
- Clean white background for content areas
- Green category banners for exercise titles
- Colored stat cards on the admin panel: blue, green, orange, red
- Pink/coral "Son Egzersiz" button at the bottom of exercise pages
- Alert dialog with "Merhaba, Hazırmıyız?" greeting
- All text in Turkish

**Screens**
- **Home Screen**: Shows the current exercise with category banner, image, description, and action buttons
- **Information Form**: Scrollable consent form explaining the study, its purpose, and procedures
- **Exercise Detail**: Full exercise view with image, title, instructions, and "Son Egzersiz" button
- **Admin Panel**: Dashboard with 4 colored stat cards (Videos, Blog, Users, Support) and a user list below
- **Profile**: User account information page

**App Icon**
- Green background with white lungs/breathing symbol and "KOAH" text
